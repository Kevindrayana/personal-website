# iNotes

iNotes is a personal project built as a full-stack clone of a general note-taking app. It includes all the essential features of a note-taking app and more. The app is built using the popular MERN stack, comprising MongoDB, Express.js, React.js, and Node.js, making it fast and reliable.

## Features of iNotes

- **User Authentication**: iNotes has a secure user authentication system that ensures all your notes are private and only accessible to you. You can sign in to access your notes.

- **Create, Read, Update, and Delete Notes**: You can add a title and content to your notes, and they will be saved automatically. You can edit or delete your notes at any time. The app also sort your notes by date, making it easier to find the note you want.

- **Search Notes: With iNotes**, you can search for your notes by keywords, making it effortless to find the note you need.

## Technology Stack

iNotes is built using the MERN stack, which means that it uses MongoDB for its database, Express.js for its backend, React.js for its frontend, and Node.js for its server. When you create a note, the data is sent to the server, which then saves it to the database. When you request a note, the server retrieves the data from the database and sends it to the frontend, where it is displayed.

## Conclusion

iNotes is a simple, yet powerful note-taking app that allows you to create, read, update, and delete notes with ease. It has a user-friendly interface, making it easy to use and navigate. The app is built using the MERN stack, which makes it fast and reliable.
