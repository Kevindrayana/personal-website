# iNotes

iNotes is a personal project built as a full-stack clone of a general note-taking app. It includes all the essential features of a note-taking app and more. The app is built using the popular MERN stack, comprising MongoDB, Express.js, React.js, and Node.js, making it fast and reliable.

## Features of iNotes

#### User Authentication
iNotes has a secure user authentication system that ensures all your notes are private and only accessible to you. You can create an account and sign in to access your notes. The app uses JSON Web Tokens (JWT) for authentication, ensuring your login credentials are secure.

#### Create, Read, Update, and Delete Notes
iNotes allows you to create, read, update, and delete notes with ease. You can add a title and content to your notes, and they will be saved automatically. You can edit or delete your notes at any time. The app also allows you to sort your notes by date, making iteasier to find the note you want.

#### Search Notes
With iNotes, you can search for your notes by keywords, making it effortless to find the note you need. The search feature also allows you to filter notes based on their date of creation or modification, making it easier to locate notes created or updated within a specific time frame.

#### Responsive Design
iNotes has a responsive design that ensures it works seamlessly on any device. Whether you're using a desktop, tablet, or mobile device, the app adapts to your screen size, making it easy to use and access your notes.

## How iNotes Works

iNotes is built using the MERN stack, which means that it uses MongoDB for its database, Express.js for its backend, React.js for its frontend, and Node.js for its server. When you create a note, the data is sent to the server, which then saves it to the database. When you request a note, the server retrieves the data from the database and sends it to the frontend, where it is displayed.

## Conclusion

iNotes is a simple, yet powerful note-taking app that allows you to create, read, update, and delete notes with ease. It has a user-friendly interface, making it easy to use and navigate. The app is built using the MERN stack, which makes it fast and reliable. With iNotes, you can organize your notes and access them from anywhere, ensuring that you never forget anything important.
