# Big Two Game

A client-server Big Two card game with a GUI. Big Two is a game of skill and strategy that is easy to learn but challenging to master. The game can accommodate up to four concurrent players, making it a perfect choice for a fun game night with friends.

## Features of Big Two Game

#### Multiplayer Support:

Big Two Game can accommodate up to four concurrent players. Players can join a game by entering the server's IP address and port number. Once connected, players can create a new game or join an existing one.

#### Gameplay Mechanics:

The objective of Big Two Game is to be the first player to get rid of all their cards. Each player is dealt 13 cards, and the player with the 3 of diamonds goes first. Players must play a card of the same suit or a higher rank to beat the previous player's card. The game continues until one player has no cards left.

#### Chat System:

Big Two Game features a chat system that allows players to communicate with each other during the game. This feature adds a social element to the game, making it more fun and interactive.

## How Big Two Game Works

Big Two Game is a client-server application built using Java programming language. The client-side of the game features a responsive GUI built using Java Swing, while the server-side of the game is built using threads and sockets.

## Conclusion

Big Two Game is an exciting and challenging multiplayer card game that is perfect for game nights with friends. Its responsive GUI and multiplayer support make it accessible on a wide range of devices, while its gameplay mechanics and chat system make it more engaging and interactive. The game is built using Java programming language, which makes it easy to customize and extend.
