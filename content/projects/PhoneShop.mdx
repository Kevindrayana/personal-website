# Phone Shop

Phone Shop is a single-page application (SPA) built on Angular, modeling a general e-commerce site front end. It includes all the essential features you would expect from an e-commerce site and more.

## Features of Phone Shop

- **Toggle Item Details**: Phone Shop allows users to click on an item to toggle its details, including the item's name, image, description, and price. This feature provides users with more information about the product

- **Add Items to Cart**: Users can add items to their cart by clicking on the "Add" button next to each item. This feature makes it easy for users to keep track of their purchases.

- **Pagination**: Phone Shop includes a pagination feature that allows users to slide between pages of items by clicking on the arrows at the bottom of the page. This feature makes it easy for users to browse through a large number of products without feeling overwhelmed.

## Conclusion

Phone Shop is a great e-commerce site that offers all the essential features you would expect from an online store. Its user-friendly interface makes it easy for anyone to purchase products online. The app is built on Angular, which is a popular JavaScript framework for building web applications. Its toggle item details, add items to cart, and pagination features make it easy for users to browse through a large number of products without feeling overwhelmed.
